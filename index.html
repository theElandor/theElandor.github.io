1.2 Sfocatura di un immagine (17/03/22)

Vengono di seguito riportati i passi fondamentali:

  • Applico una matrice data (per effettuare la sfocatura) all' array ottenuto dalla matrice originale incolonnando tutti gli elementi
  • Stampo l'immagine e mi rendo conto di aver effettuato la sfocatura;
  • Provo a risolvere il sistema \(Hx=b\) per vedere se dall'immagine sfocata riesco a ottenere l'immagine originale(ossia x);
  • Mi rendo conto che l'immagine ottenuta è abbastanza fedele all'originale;
  • Cerco di capire se il problema è mal-condizionato / ben-condizionato: vedo se applicando una perturbazione ai dati di partenza (ossia all'immagine sfocata) riesco lo stesso ad ottenere una soluzione fedele all'immagine originale.
  • Stampando la soluzione del sistema partendo dai dati perturbati mi rendo conto che l'immagine è totalmente distorta: basta un piccolo cambiamento per rendere il sistema "quasi irrisolvibile."
  • Il problema è mal condizionato.
[n,m] = size(obj);
x = obj(:); % mette tutti gli elementi di una matrice in colonna
b = H*x; % applico la sfocatura
b_immagine=reshape(b,n,m); % crea dagli elementi di B una matrice nxm
figure
imshow(b_immagine,[]); %immagine sfocata
sol1 = H\b; %il vettore colonna che esce è la approssimazione di x, 
	    %soluzione del sistema lineare.
sol1_immagine = reshape(sol1,n,m);
figure
imshow(sol1_immagine,[]); % immagine che provo a restorare partendo dalla
			  % sfocatura
Deltab = randn(n*m,1); %applico un errore molto piccolo, se stampo la
		       %figura non si vede nemmeno la sfocatura
b2 = b +Deltab*1e-8;
b2_immagine = reshape(b2,n,m);
figure
imshow(b2_immagine, []);

sol2 = H\b2; % risolvo il sistema Hx=b2, ignorando il rumore
sol2_immagine = reshape(sol2,n,m);
figure
imshow(sol2_immagine, []);
